Ejercicio Integrador 1:  Ejercicio que integra View-Controller-Model, Formatter, Constants, I18n, Dialog & Fragment

El cliente necesita una APP que muestre una tabla con productos y un buscador:
  
La misma deberá estar vacía hasta que el usuario ingrese lo parámetros de búsqueda y presione “Ir”
  
El buscador deberá tener tres campos: Fecha, Proveedores (select option) y País (tipo input libre)
  
Al presionar “Ir” se vera un mensaje (SAPUI5) con los datos ingresados y se cargara la tabla con los datos obtenidos de un json local 
Fecha: “fecha ingresada”

Proveedor: “proveedor seleccionado”

País: “país ingresado”
  
El formato del json deberá ser:
Productos:

            Producto

            Proveedor

            Tamaño

            Peso

            Valor 
  
La misma deberá “pintar” el valor del campo peso según la siguiente regla:

                        Peso < 1Kg  Correcto

                        Peso > 1kg y Peso < 2kg  Advertencia

                        Peso > 2kg   Peligro
  
El valor obtenido desde el “Back End” en pesos argentinos deberá traducirse a euros (valor en pesos /160)

La tabla deberá contar con un buscador que busque por producto y proveedor
  
La aplicación se deberá ejecutar tanto en español como en ingles (español por defecto)
  
Al presionar sobre un elemento se deberá abrir un dialogo mostrando la información de este
    
Ayuda para obtener la información de un ítem seleccionado:

var oItem = oEvent.getSource();

var oBindingContext = oItem.getBindingContext("nombreDelModelo");
